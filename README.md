SAF is a Chrome extension designed to make testing Stripe checkout forms easy and efficient. It comes equipped with a variety of features that automate form filling, generate card information, and manage addresses, all tailored to streamline your testing process.

## üöÄ Features

- **Automatic Form Filling:** Quickly fill Stripe checkout pages without manual input.
- **Card Number Generation:** Create valid card numbers with Luhn validation.
- **BIN Template Support:** Use cards of varying lengths (6-19 digits).
- **Address Generation:** Automatically generate addresses based on IP geolocation.
- **Custom Address Management:** Store and manage multiple addresses and names.
- **Multi-Theme User Interface:** Choose from Dark, Light, Galaxy, Sky, or Underground themes.
- **IP Blocker:** Prevent detection during 3DS challenges.
- **Local Data Storage:** No external servers; all your data stays private.
- **BIN History:** Easily access previously used BINs.

4. **Load the Extension in Chrome:**

   - Enable **Developer mode** by toggling the switch in the top right corner.
   - Click on **Load unpacked.**
   - Select the folder where you downloaded the SAF extension.

5. **Check the Extension:** Look for SAF in your list of Chrome extensions and ensure it is enabled.
